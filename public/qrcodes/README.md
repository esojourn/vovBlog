# QR Code Directory

Place WeChat official account QR codes here.

## Naming Convention

- `wqws.png` - "瓦器微声" QR code
- `yds.png` - "盐读书" QR code
- `wbey.png` - "五饼二鱼能量站" QR code

## File Format

- Format: PNG or JPEG
- Recommended size: 200x200px minimum
- Files are referenced in `lib/source-config.ts` with paths like `/qrcodes/wqws.png`

## Usage

These QR codes are configured in `lib/source-config.ts` under the `qrCode` field of each source.
They can be displayed in the UI or used for WeChat following links.
