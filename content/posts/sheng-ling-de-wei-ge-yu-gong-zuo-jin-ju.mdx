我们自己不能使人知罪，神也没有把这工作留给我们去做。祂把这工作交在一位更有能力的手中，就是圣灵。\
\
无论我们的讲道多么合乎正统，无论我们如何研读神的话，除非圣灵动工，否则都不能重生人。\
\
除非圣灵亲自以此见证，否则没有人会真正看见耶稣并相信。\
\
圣灵在人心中的工作不是要荣耀祂自己，而是要荣耀基督。\
\
\
真正的基督徒生活不是由外在的一长串规则所管理的，而是由内在的一位活泼且永远同在的位格所引导的。\
\
当我们领受圣灵时，我们就领受了神自己的生命，以一种位格的方式住在我们里面。\
\
如果圣灵在我们里面成形了基督，我们所要做的就是让这位内住的基督在我们里面活出祂自己的生命。\
\
被圣灵引导并不意味着我们会做神写在书上的话语告诉我们不要做的事。圣灵绝不会引导人违背祂自己所默示的圣经。\
\
\
如果圣灵是一个位格，而且是神圣的位格，而我们不以此来认识祂，那么我们就是抢夺了神圣位格所当得的崇拜、信心、爱和顺服。\
\
我们若按圣经的方式视祂为神圣的位格，我们的想法就会是：“圣灵如何能更多地拥有我？”\
\
圣灵是神在我们里面的“驻防者”或“帮助者”，如果我们只要信靠祂去做，祂就会以此来掌管并帮助我们。\
\
不要叫神的圣灵担忧。如果我们将这句话铭刻在心，并以此作为我们生活的座右铭，它将保守我们不犯许多的罪。\
\
\
不信是所有罪中最难使人知罪的。普通的不信者并不认为不信是罪，甚至以此为智力优越的标记。\
\
当神的灵触摸一个人的心，他就不再视不信为智力优越的标记，而将其视为最鲁莽、最决定性、最该受咒诅的罪。\
\
\
如果风在一小时内完全停止吹动，这地上的大部分生命将会停止。照样，如果圣灵在任何一颗心、教会或社区中停止吹气，死亡就会随之而来。\
\
我们应当像相信我们若求饼，父亲不会给我们石头一样，相信只要我们求，天父就必将圣灵赐给我们。\
\
每一个真信徒都有两位神圣的位格在为他祷告：那就是在天上已经升入高天尊荣里的神子，和在地上住在最卑微的信徒心中的圣灵。\
\
若不是圣灵那种恒久忍耐、长久受苦、从不厌倦、无限温柔的爱，我今天早就在地狱里了。\
\
若这关于圣灵是保惠师的真理一旦进入你的心并住在那里，它将永远驱逐所有的寂寞。\
\
\
《圣灵的位格与工作》问答：\
\
如果我们将圣灵仅仅视为一种神圣的影响力或力量，会对我们的属灵生命产生什么根本性的误导？ 如果我们仅仅视圣灵为一种力量，我们会产生野心和骄傲，试图去获得并利用这种力量来达成自己的目的；但如果我们视祂为一位拥有无限威严的神圣位格，我们就会谦卑俯伏，思考如何让祂来完全拥有并使用我们，这种观念的转变决定了我们是走向自我膨胀还是在尘土中谦卑同行。\
\
有些人认为只要读了圣经就能明白真理，为什么书中强调必须要有圣灵的教导才能真正领悟神的话语？ 虽然圣经是神的话语，但属血气的人无法领会神圣灵的事，因为这些事唯有属灵的人才能看透。就像欣赏艺术需要审美能力一样，理解属灵的启示需要圣灵的照明；单靠人的理智和逻辑研究圣经，只能得着字句的知识，唯有圣灵亲自解释祂所默示的真理，人才能明白其中深奥的属灵实际。\
\
为什么作者说“顺服”是领受圣灵之洗的关键步骤，甚至比祷告更为决定性？ 神将圣灵赐给顺从祂的人，顺服意味着意志的完全降服。许多人虽然恳切祷告，但在内心深处仍保留着某个不愿意交出的“小房间”或某种不愿意放弃的罪，这种有保留的态度拦阻了圣灵的充满。只有当我们将自己作为全牲的燔祭献在祭坛上，毫无保留时，神的火才会降下悦纳这祭物。\
\
如果耶稣基督是神的儿子，拥有神性，为什么祂在世上的生活和事奉还需要完全依赖圣灵？ 这是为了显明祂完全的人性，并为我们留下榜样。耶稣倒空自己，取了奴仆的形象，祂并非靠自己内在的神性能力行事，而是作为一个完全依靠神的人，藉着圣灵的能力去得胜试探、赶鬼、行神迹和教导。这告诉我们，既然神的儿子尚且需要依赖圣灵，我们这些跟随者更离不开圣灵的大能。\
\
在寻求圣灵引导的过程中，信徒如何分辨是出于圣灵的声音，还是出于自己的私欲或邪灵的欺骗？ 圣灵绝不会引导人去做违背圣经明文教导的事，因为祂不能背乎自己。真正的引导必须经过神话语的检验，并且是在信徒意志完全顺服神、不从私欲出发的前提下，通过圣灵微小的声音、环境的印证以及心中属天的平安来显明的。\
\
有人因为没有说方言或没有强烈的情感体验而怀疑自己是否受了圣灵的洗，这种看法符合圣经吗？ 不符合。圣经教导圣灵随己意将恩赐分给各人，岂都是说方言的吗？圣灵受洗的主要凭据是得着为主作见证的能力，而非某种特定的恩赐或情感的狂喜。过分追求外在现象或感觉，反而可能导致错失信心，因为我们是凭着神的应许而非凭着感觉来确信自己已得着。\
\
书中提到的“住在里面的圣灵”如何解决现代人普遍面临的孤独与空虚感？ 耶稣应许圣灵要成为另外一位保惠师，永远与我们同在。这意味着信徒拥有一个内在的、永不枯竭的满足源头，不再依赖外在的环境、财富或人际关系来获得快乐。当人认识到这位全能、慈爱且全知的朋友时刻居住在自己心灵的圣所中时，孤独便被神圣的陪伴所取代，空虚被活水的泉源所充满。\
\
既然圣灵在五旬节已经赐下给教会，为什么我们今天还需要祈求圣灵的充满？ 五旬节是圣灵作为恩赐赐给教会整体的历史性时刻，但每一个体信徒仍需凭信心去支取这一恩典。就像救恩已在十字架上完成，但每个人仍需通过悔改信主来得着一样。此外，经文显示门徒在五旬节后仍多次被圣灵充满，表明这是一次性的得着（受洗）与持续性的更新（充满）相结合的过程，我们在面临新的事奉挑战时，需要神新鲜的能力。\
\
在圣经中，既然所有的信徒都有圣灵内住，为什么作者还强调“受圣灵的洗”是一个独特的、额外的经验？ 重生是圣灵将神的生命分赐给人，使人得救；而圣灵的洗是圣灵将属天的能力赋予人，使人得着事奉的恩赐。根据使徒行传的记载，如撒玛利亚的信徒和以弗所的门徒，他们虽然已经信主重生，但尚未经历圣灵降在他们身上赋予能力的特殊时刻，这表明在生命得救与得着事奉能力之间，可能存在着经验上的区分。\
\
\
《圣灵的位格与工作》作者：\
\
鲁本·阿彻·叨雷（Reuben Archer Torrey, 1856–1928）是19世纪末至20世纪初美国著名的布道家、牧师、神学家和教育家。他出生于新泽西州的一个富裕家庭，早年毕业于耶鲁大学和耶鲁神学院。在大学期间，他曾经历过深刻的怀疑主义危机，但最终在一种超自然的经历中彻底归向神，这为他日后强调圣灵工作和祷告神学奠定了基础。\
\
叨雷是著名布道家D.L. 慕迪（D.L. Moody）的亲密同工和得力助手。1889年，他被任命为芝加哥圣经学院（即现在的慕迪圣经学院）的首任院长，对确立该学院的教育方针和神学立场起到了关键作用。随后，他还担任了芝加哥大道教会（现为慕迪教会）的牧师。1912年至1924年间，他担任洛杉矶圣经学院（现为拜欧拉大学，Biola University）的院长，并创立了洛杉矶的敞开之门教会（Church of the Open Door）。\
\
作为一位布道家，叨雷与查尔斯·亚历山大（Charles Alexander）搭档，在1902年至1906年间进行了环球布道旅行，足迹遍布英国、澳大利亚、新西兰、印度、中国和日本等地，据统计有超过十万人因他们的事工决志信主。他在中国的布道对当时的中国教会产生了深远影响。\
\
叨雷是一位多产的作家，著有四十多部书籍，其中《圣灵的位格与工作》、《如何领人归主》、《如何祷告》等皆为基督教经典之作。他是基要主义神学的重要奠基人之一，曾主编《基要真理》（The Fundamentals）丛书，捍卫圣经无误论和基督的神性。他的神学思想以圣经为绝对权威，特别强调圣灵的洗作为事奉能力的来源，这一教导对后世的五旬节运动和灵恩运动产生了一定的启蒙影响，尽管他本人在神学立场上保持着传统的福音派色彩。
